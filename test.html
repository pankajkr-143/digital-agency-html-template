Project: To-Do List Application

Objective: Build a functional to-do list where users can add, remove, and mark tasks as completed. This project will include responsive design and basic interactivity.

Features

  • Add Tasks: Users can type a task into an input box and click a button to add it to their to-do list.
  • Mark as Completed: Users can click on a task to mark it as completed. Completed tasks will have a strikethrough.
  • Remove Tasks: Users can delete tasks by clicking a delete icon next to each task.
  • Responsive Design: The application should look good on both desktop and mobile.

File Structure

  • index.html: Contains the HTML structure.
  • style.css: Adds styling to the application.
  • app.js: Contains the JavaScript to add functionality.
